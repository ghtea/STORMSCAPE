## 개요
- 게임 캐릭터의 단기간 통계수치를 이용해서, 각 경우별 승률을 시각화 해서 웹페이지로 표현
  - 각 캐릭터의 전체 승률, 다른 특정 캐릭터와 같은 팀일 때의 승률, 상대팀일 때의 승률, 맵별 승률, 특정 아이템을 선택했을 때의 승률
- 모바일 페이지로 디자인
- api 비용 관계로 배포 중지
